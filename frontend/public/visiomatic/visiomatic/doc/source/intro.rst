============
Introduction
============

The purpose of this manual is to help administrators and data scientists setting
up and operating |VisiOmatic|_ and |IIPImage-astro|_ together.

This manual is divided in four main sections. The first section gives a technical 
overview of |IIPImage| and |VisiOmatic|, and their capabilities. The second
section will guide administrators through installation and configuration of
|IIPImage-astro|. The third section explains how to set up and customize the
|VisiOmatic| web interface through concrete examples. Finally, the last section
provides a full description of the web client
`API <http://en.wikipedia.org/wiki/Application_programming_interface>`_.
